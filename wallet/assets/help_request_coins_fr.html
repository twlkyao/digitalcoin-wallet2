En utilisant cette boîte de dialogue, vous pouvez recevoir des digitalcoins (vous devez avoir installé digitalcoin Wallet).

D'abord, saisissez le montant de digitalcoins à recevoir.

Ensuite, l'autre personne doit scanner le code QR généré. Ou bien, vous pouvez envoyer une URL digitalcoin à une personne à distance avec le bouton "publier" dans la barre d'action. L'autre personne devra cliquer l'URL digitalcoin.

Dans les deux cas, l'autre personne se verra présenter une boîte de dialogue pour envoyer des digitalcoins avec tous les champs pré-remplis. Si elle accepte, elle n'a plus qu'à envoyer le paiement.
